# Sample Local Document

This is a placeholder file so you have something to point `DOCUMENT_INPUT_DIR`
at on your first run. Replace it with your own notes, PDF, or markdown file —
anything you want the Research Crew to pull into the guide.

Local file paths passed to the flow must resolve inside this `documents/`
directory (see the path-traversal check in `validate_inputs`), so keep any
files you want to use here or in a subdirectory of it.
